# homebrew-tap

Official Homebrew tap for [Eresus Security](https://eresussec.com) tools.

## Install

```bash
brew tap eresus-security/tap
brew install sentinel
```

## Update

```bash
brew update
brew upgrade sentinel
```

## Available Formulae

| Formula | Description |
|---|---|
| `sentinel` | Eresus Sentinel — Deterministic AI security toolkit |

## Uninstall

```bash
brew uninstall sentinel
brew untap eresus-security/tap
```

## Manual Install (without tap)

```bash
brew install https://raw.githubusercontent.com/eresus-security/homebrew-tap/main/Formula/sentinel.rb
```

---

> Formula updates are automated by CI on every release of [Eresus-sentinel](https://github.com/EresusSecurity/Eresus-sentinel).
